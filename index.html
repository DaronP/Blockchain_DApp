Description Of the Project

The DApp is a system that works with the best technology. A Smart contract is implemented that is responsible for interacting with the system. This guarantees the smooth operation of the application and the interaction of buying coins. It is intended to create a space where the user can interact with the currency and thus obtain the expected results according to the operation indicated by the client.

Steps

Sysrem analysis - Create PepeCoin

The "PepeCoin" coin is created for the Blockchain project for the Course "Advanced Topics of Computing 2" at the Universidad del Valle de Guatemala. The students establish what rules the currency will have and what transactions will be possible for this scope of the system.

Fast interaction

The smart contract guarantees a quick and immediate interaction with the system. The connection is established through functions that are responsible for providing the user with a prompt response without interaction from other media.

Create the public site

A public site with a responsive interaction was prepared to the client of the system.

Premium Content
